## ğŸ“– Tentang Proyek

**Sistem Pakar Kerusakan Sepeda Motor** adalah aplikasi berbasis web/desktop yang dirancang untuk membantu mekanik pemula maupun pengguna awam dalam mendiagnosa kerusakan pada sepeda motor. 

Sistem ini bekerja dengan meniru kemampuan pengambilan keputusan seorang pakar (mekanik ahli). Pengguna cukup memilih gejala-gejala yang dialami motor, dan sistem akan memberikan hasil diagnosa beserta solusi perbaikannya.

### ğŸ§  Metode yang Digunakan
Sistem ini menggunakan metode **[Sebutkan Metode: misal Forward Chaining / Certainty Factor / Dempster Shafer]** untuk menelusuri fakta-fakta (gejala) menuju kesimpulan (kerusakan).

Berikut adalah fitur utama dari aplikasi ini:

### ğŸ‘¤ Halaman Pengguna (User)
- **Diagnosa Interaktif:** Memilih gejala kerusakan melalui antarmuka yang mudah.

- **Riwayat Diagnosa:** Menyimpan history pengecekan sebelumnya.
- **Cetak Laporan:** Fitur untuk mencetak hasil diagnosa ke PDF.

- **Dashboard Admin:** Statistik penggunaan dan data diagnosa.

* [Misal: Composer / Pip]

### Langkah Instalasi

1. **Clone Repositori**
